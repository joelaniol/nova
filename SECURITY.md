# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in Nova Browser, please report it responsibly via [LinkedIn](https://www.linkedin.com/in/joelaniol/).

Do not open a public issue for security vulnerabilities.

## Supported Versions

| Version | Status |
|---|---|
| 1.0.0-beta | Current beta — actively maintained |

## Beta Software

Nova Browser is beta software. While security is a core design goal, bugs and vulnerabilities may exist. Use in security-critical environments is at your own risk.

# Sicherheitsrichtlinie

## Schwachstellen melden

Wenn du eine Sicherheitslücke in Nova Browser findest, melde sie bitte vertraulich über [LinkedIn](https://www.linkedin.com/in/joelaniol/).

Bitte keine öffentlichen Issues für Sicherheitslücken erstellen.

## Beta-Software

Nova Browser ist Beta-Software. Obwohl Sicherheit ein zentrales Designziel ist, können Fehler und Schwachstellen existieren. Die Nutzung in sicherheitskritischen Umgebungen erfolgt auf eigenes Risiko.
